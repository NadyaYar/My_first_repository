Git - система контроля версій файлів .
Statuses commited modified staged untracked. 
Команда add використовується для додавання файлів.
Команда status використовується для перевірки статуса файлів.
Команда commit використовується для додавання файлів в локальний репозиторій.
Команда init використовується для ініціалізації git н проектю
У даній команді ми повинні писати інструкцію що ми змінюємо у commit .
Для того щоб переглянути список commit , ми повинні використати "git log --reflog"
Команда checkout допомагає запустити зміни у файлі в потрібну ветку
